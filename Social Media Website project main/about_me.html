About Me

Hi, my name is Flosie Sazon. I am a 19 years old Creative Digital Media student in Technological University Dublin (TU Dublin) located in Blanchardstown Campus. I was born in the Philippines and moved to Ireland in 2013 to continue my studies.

I created this website especially for students and teachers that are not familiar with different social media tools that they can use for education. This website has 5 different applications that you might decide to use after reading the different types of information I gathered such as their basic information, instructions on how to use the App with a video and also its cautions and safety. I hope you find my website helpful and If you want to share your opinions or you have more questions, feel free to ask me by signing up an application form in the contact page!

Mission

My mission for this website is to provide enough information about social media tool to help my client understand clearly. This website should have an accurate demonstration on how each tools work and can benefit them if they will use it for learning or teaching purposes. Lastly, the website must be well-designed so that the client can navigate it easily and can access it to any devices without any complication.

Goals

My goals for this website are to have a neat template on each of my html page and a stable web design. I also want to achieve all the objectives that I set before creating this website and to satisfy my client by providing all the necessary data he/she needed.
